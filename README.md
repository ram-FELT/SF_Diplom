# SF_Diplom
Запуск:

1) в дирректории pip install -r requirements.txt  **(Терминал_1)

2) \Silant\frontend> npm install                  **(Терминал_2)

3) \Silant> python manage.py runserver            **(Терминал_1)

4) \Silant\frontend> npm run dev                  **(Терминал_2)
